# Business

Module to generate business related data.

## Overview

Generate a random business category.

## category

Returns a random business category.

***Returns***: string

```rust
use data_faking::locales::en as faking;

fn main() {
    println!("{}", faking::business::business::category())
}

//Aerospace company
```
