# Commerce

Module to generate commerce related data.

## Overview

Generate a random appliace brand or equipment.

## department

Returns a random commerce department.

***Returns***: string

```rust
use data_faking::locales::en as faking;

fn main() {
    println!("{}", faking::business::commerce::department())
}

//Books
```
