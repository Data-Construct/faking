# Industry Segments

Module to generate data about industry segments.

## Overview

Capable of generating data about industry segments.

## industry

Returns a random industry type.

***Returns***: string

```rust
use data_faking::locales::en as faking;

fn main() {
    println!("{}", faking::business::industry_segments::industry())
}

//Oil & Gas
```
